## 1. 프로젝트 개요

### 1.1 프로젝트 코드명
**(미정)** — 후보: NextMe / DayCurator / Briefly / 하루비서

### 1.2 한 줄 정의
> **사용자는 생각 없이 기록만, AI가 정리하고 실행까지 연결해주는 개인 비서 앱**

### 1.3 비전
사용자가 더 적게 고민하고, 더 많이 실행하고, 매일 성장하는 삶을 만든다.

### 1.4 미션
음성/텍스트로 흘려보내는 일상을 AI가 구조화하여, 일상과 비즈니스를 균형 있게 관리하는 자동화된 개인 비서 시스템을 제공한다.

### 1.5 핵심 가치 제안 (UVP)
1. **압도적 시간 절약**
2. **실행력 극대화**
3. **일상 & 비즈니스 균형**
4. **진짜 맞춤형 비서**

### 1.6 타겟 사용자
- **Primary**: 다중 영역 관리가 필요한 30~50대
- **Secondary**: 자기계발 의지가 있는 25~40대 직장인/창작자

## 2. 시스템 구성 (확정) — 모바일 앱 + 어드민 콘솔

### 2.0 시스템 구조 개요

이 프로젝트는 **두 개의 독립된 클라이언트**로 구성된다.

```
┌──────────────────────────┐    ┌──────────────────────────┐
│   📱 모바일 앱 (메인)    │    │   🖥️ 어드민 콘솔 (운영)  │
│   React Native + Expo    │    │   Next.js (Web)          │
│   사용자용 (iOS/Android) │    │   본인/운영팀 전용       │
└────────────┬─────────────┘    └────────────┬─────────────┘
             │                                │
             └──────────┬─────────────────────┘
                        ↓
            ┌──────────────────────────┐
            │  ☁️ Firebase 백엔드      │
            │  Auth / Firestore /      │
            │  Functions / Storage /   │
            │  FCM                     │
            └──────────────────────────┘
                        ↓
            ┌──────────────────────────┐
            │  🤖 AI / 외부 서비스     │
            │  Gemini Flash-Lite/Flash │
            │  Google Calendar API     │
            └──────────────────────────┘
```

### 2.1 모바일 앱 — 프론트엔드
- **프레임워크**: React Native + Expo (최신 SDK)
- **언어**: TypeScript
- **상태 관리**: Zustand
- **UI 스타일링**: NativeWind (Tailwind CSS for RN)
- **애니메이션**: React Native Reanimated 3
- **네비게이션**: Expo Router (파일 기반)
- **국제화**: i18next + react-i18next
- **다크모드**: NativeWind 다크모드

### 2.2 어드민 콘솔 — 프론트엔드 (NEW)
- **프레임워크**: Next.js 14 (App Router)
- **언어**: TypeScript
- **UI**: shadcn/ui + Tailwind CSS
- **차트**: Recharts (통계 대시보드)
- **데이터 페칭**: SWR or TanStack Query
- **인증**: Firebase Auth + Custom Claims (admin role)
- **배포**: Vercel
- **도메인**: `admin.{앱이름}.com`
- **접근 제어**: 2FA 필수, 화이트리스트 IP (선택)

### 2.3 백엔드 (Firebase 공유)
- **인증**: Firebase Auth — Google/Apple 소셜 로그인 (앱) + 어드민 전용 로그인
- **데이터베이스**: Firestore (실시간 + 오프라인 지원)
- **서버리스 함수**: Cloud Functions — AI 파이프라인, 결제 검증, 협업 권한, **한도 체크 로직**
- **파일 저장**: Cloud Storage (음성 7일, 이미지 영구)
- **푸시 알림**: FCM
- **분석/모니터링**: Firebase Analytics + Crashlytics

### 2.4 AI 모델 (확정 — 다중 라우팅)

| Tier | 모델 | 가격 (1M 토큰) | 용도 |
|---|---|---|---|
| **Tier 1** | Gemini 2.5 Flash-Lite | $0.10 / $0.40 | 분류, 단순 추출 |
| **Tier 2** | Gemini 2.5 Flash | $0.30 / $2.50 | Draft 생성, 챗봇, 리포트 |

#### 작업별 라우팅
- **Tier 1**: 분류, 키워드/엔티티 추출, 우선순위 자동 분류, 텍스트 정제, 감정 태그
- **Tier 2**: Draft 생성, 챗봇 응답, 하루 리포트, 회고 인사이트, 만다라트 구조 제안, Top 3 추천, 맥락 리마인드, 다국어 응답

#### 예상 월 비용 (1,000명 기준)
- 표준 비용: ~$400/월
- 캐싱 90% 적용 시: ~$150/월

### 2.5 외부 연동
- **STT**: 디바이스 네이티브 (1차) + Whisper API (백업)
- **캘린더**: Google Calendar API + Apple Calendar (EventKit)

### 2.6 결제 (정식 출시 단계)

- **Android 인앱결제**: Google Play Billing

### 2.7 배포 & 운영

## 5. Tier 시스템 (NEW)

### 5.1 등급 구조 (가정값 — 베타 후 확정)

| Tier | 코드 | 음성/회 | 음성/일 | 텍스트 | Project | 협업 |
|---|---|---|---|---|---|---|
| **Free** | `FREE` | 1분 | 10회 | 1,000자 | 1개 | 0명 |
| **Basic** | `BASIC` | 5분 | 50회 | 5,000자 | 5개 | 3명 |
| **Pro** | `PRO` | 30분 | 무제한 | 10,000자 | 무제한 | 10명 |
| **Enterprise** | `ENT` | 무제한 | 무제한 | 무제한 | 무제한 | 무제한 |

### 5.2 정책
- **무료 사용자도 풀 기능 사용 가능** (단, 한도만 제한)
- 한도 초과 시 안내 + 업그레이드 유도
- 어드민이 사용자별 한도 수동 조정 가능 (베타 테스터, VIP, 지원 케이스)
- 결제 모델 활성화는 Phase 4 (정식 출시) 시점

### 5.3 데이터 흐름
```
사용자 행동 (음성 녹음 등)
  → Cloud Functions: 사용자 한도 조회 (override 우선, 없으면 tier 기본값)
  → 오늘 사용량 조회
  → 한도 초과 시 → 차단 + 업그레이드 유도
  → 한도 내면 → 진행 + 사용량 카운트 증가
```

## 6. 핵심 사용자 시나리오

### 6.1 아침 (06:00~09:00)
1. 사용자 기상 → 앱 열기
2. AI(Flash)가 "오늘 우선순위 Top 3" 제시
3. 오늘 일정 미리보기
4. 푸시: "오늘의 우선순위를 확인하세요"

### 6.2 낮 (09:00~18:00)
1. 음성/텍스트 즉석 기록
2. AI(Flash-Lite)가 LIFE/WORK 분류 → AI(Flash)가 Draft 생성
3. 사용자 확인 → 확정 → ToDo + 캘린더 자동 등록
4. 만다라트 연결 (선택)

### 6.3 저녁 (18:00~21:00)
1. 진행률 표시 (예: 75%)
2. 미완료 리마인드 (맥락 기반)
3. 푸시: "오늘 마지막 할 일 어때요?"

### 6.4 밤 (21:00~23:59)
1. 하루 리포트 (감정 요약, 성과 분석)
2. 내일 우선순위 Top 3 추천
3. 푸시: "잘했어요! 꾸준함이 최고의 성장입니다"
