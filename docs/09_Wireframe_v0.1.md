**버전:** v0.1
**작성일:** 2026-04-27
**상태:** 초안

## 0. 문서 개요

본 프로젝트의 모든 화면 와이어프레임 + 디자인 시스템 정의.

**기준 문서**: User Flow v0.1 / PRD v0.6

**다루는 범위:**
- 디자인 시스템 (색상, 타이포, 간격, 컴포넌트)
- 모바일 앱 화면 (Phase 1~4)
- 어드민 콘솔 화면
- 컴포넌트 카탈로그
- 반응형 / 다크모드 가이드

**표기법:**
- 모바일 화면 너비 36자 = 약 360pt (실제 360~414pt 범위)
- `[버튼]` = 탭 가능
- `( )` = 라디오 / `[ ]` = 체크박스
- `▼` = 드롭다운 / `→` = 화살표 진입
- `═` = 강조 영역 / `─` = 일반 구분

```
none:   0
```

## 2. 네비게이션 구조

### 2.1 모바일 — 5탭 + 중앙 FAB

```
┌────────────────────────────────────┐
│        화면 콘텐츠 영역             │
│                                    │
│                                    │
│                                    │
│                                    │
│                                    │
└────────────────────────────────────┘
┌────────────────────────────────────┐
│  🏠     ✓     ⊕    📅     ⚙️       │
│  홈    할일   FAB  캘린더  설정      │
└────────────────────────────────────┘

탭 1: 홈
탭 2: 할일 (ToDo)
중앙 FAB: 음성/텍스트 입력 모달 (탭 X, 떠 있음)
탭 3: 캘린더
탭 4: 설정

추가 진입:
- 만다라트: 홈/ToDo에서 카드/메뉴
- 챗봇: 홈 우상단 아이콘 (Phase 2)
- 감정 타임라인: 홈 슬라이드 (Phase 2)
- 리포트: 캘린더 상단 메뉴 (Phase 2~3)
```
