### 1.1 왜 Monorepo인가?
- **타입 공유**: `Task`, `User`, `UserLimits` 등 타입 정의를 모든 앱에서 사용
- **상수 공유**: Tier 정보, 색상 팔레트 등 일관성
- **Firebase 헬퍼 공유**: 앱과 어드민이 같은 Firestore 클라이언트 사용
- **변경 추적 용이**: 한 PR에서 여러 패키지 동시 수정 가능

### 1.2 패키지 매니저: pnpm

- npm/yarn보다 빠르고 디스크 절약
- workspace 지원 우수
- 설치 명령:
  ```bash
  pnpm install                    # 루트에서 모든 의존성 설치
  pnpm --filter mobile dev        # mobile 앱 실행
  pnpm --filter admin dev         # 어드민 실행
  pnpm --filter functions deploy  # Functions 배포
  ```

### 4.3 함수 표준 패턴

```typescript
// ✅ 올바른 Cloud Function 표준
import { onCall, HttpsError } from 'firebase-functions/v2/https';
import { logger } from 'firebase-functions';
import { z } from 'zod';
import { checkUserLimit } from '@/users/checkUserLimit';
import { ClassifyInputSchema } from '@/utils/validation';

export const classifyInput = onCall(
  {
    region: 'asia-northeast3',
    cors: true,
    maxInstances: 100,
  },
  async (request) => {
    // 1. 인증
    if (!request.auth) {
      throw new HttpsError('unauthenticated', '로그인이 필요합니다');
    }

    // 2. 입력 검증
    const parsed = ClassifyInputSchema.safeParse(request.data);
    if (!parsed.success) {
      throw new HttpsError('invalid-argument', parsed.error.message);
    }

    // 3. 한도 체크
    const limit = await checkUserLimit(request.auth.uid, 'aiCallFlashLite');
    if (!limit.allowed) {
      throw new HttpsError('resource-exhausted', '한도 초과', limit);
    }

    // 4. 본 작업
    try {
      const result = await callGeminiFlashLite(parsed.data.text);
      return { ok: true, data: result };
    } catch (error) {
      logger.error('classifyInput failed', {
        uid: request.auth.uid,
        error,
      });
      throw new HttpsError('internal', '처리 실패');
    }
  }
);
```

### 8.3 에러 처리 패턴

#### 8.3.1 Result 타입 사용
```typescript
// types/result.ts
export type Result<T, E = Error> =
  | { ok: true; data: T }
  | { ok: false; error: E };

// 사용
async function getTask(id: string): Promise<Result<Task>> {
  try {
    const data = await db.collection('tasks').doc(id).get();
    if (!data.exists) {
      return { ok: false, error: new Error('Task not found') };
    }
    return { ok: true, data: data.data() as Task };
  } catch (error) {
    return { ok: false, error: error as Error };
  }
}

// 호출 측
const result = await getTask(taskId);
if (!result.ok) {

  return;
}
const task = result.data; // 타입 안전
```

### 8.5 상태 관리 (Zustand)

#### 8.5.1 스토어 정의 패턴
```typescript
// stores/tasks.ts
import { create } from 'zustand';
import { devtools, persist } from 'zustand/middleware';
import type { Task } from '@shared/types';

interface TasksState {
  // State
  tasks: Task[];
  selectedTaskId: string | null;
  isLoading: boolean;

  // Actions
  setTasks: (tasks: Task[]) => void;
  addTask: (task: Task) => void;
  updateTask: (id: string, patch: Partial<Task>) => void;
  removeTask: (id: string) => void;
  selectTask: (id: string | null) => void;
}

export const useTasksStore = create<TasksState>()(
  devtools(
    persist(
      (set) => ({
        tasks: [],
        selectedTaskId: null,
        isLoading: false,

        setTasks: (tasks) => set({ tasks }),
        addTask: (task) =>
          set((state) => ({ tasks: [...state.tasks, task] })),
        updateTask: (id, patch) =>
          set((state) => ({
            tasks: state.tasks.map((t) =>
              t.id === id ? { ...t, ...patch } : t
            ),
          })),
        removeTask: (id) =>
          set((state) => ({
            tasks: state.tasks.filter((t) => t.id !== id),
          })),
        selectTask: (id) => set({ selectedTaskId: id }),
      }),
      { name: 'tasks-storage' }
    )
  )
);
```

## 10. Git 커밋 규칙

### 10.1 Conventional Commits

```
<type>(<scope>): <subject>

<body (선택)>

<footer (선택)>
```

#### type
- `feat`: 새 기능
- `fix`: 버그 수정
- `docs`: 문서 변경
- `style`: 코드 스타일 (기능 변경 없음)
- `refactor`: 리팩토링
- `test`: 테스트 추가/수정
- `chore`: 기타 (빌드, 의존성 등)

#### scope
- `mobile` / `admin` / `functions` / `shared` / `firebase` / `docs`

#### subject
- 한국어 OK
- 명령형 ("추가했다" X, "추가" O)
- 50자 이내
- 마침표 X
